¿QUÉ SON LAS BASES DE DATOS?


En pocas palabras, las bases de datos bajo su defición técnica,

Son un conjunto de datos que se almacenan entre sí. Por dar un ejemplo,

Existen los Buffers que son muy utilizados en la creación de videojuegos

Bien sea para almacenar información dentro del juego o información para cargar

Los datos de una partida por ejemplo.


¿QUÉ ES WORDPRESS Y POR QUÉ ES IMPORTANTE?


Wordpress, creado originalmente en el 2003,

es un gestor de contenido que nos permite desarrollar sitios web

de una manera eficiente y rápida. Sin tener que crear todo el código

fuente desde 0 y haciendo así más amena la elaboración de este tipo de

sitios, pues esto permite que los clientes puedan tener control de la

página sin tener que enfrentarse directamente al código.


Cabe aclarar que otra alternativa a Wordpress es Joomla, no obstante

este último pese a que ya está.

algo obsoleto, aún hoy en día se siguen creando sitios web con él.


NOTA: UN 40% DE SITIOS WEB POR INTERNET ESTÁN CREADOS UTILIZANDO WORDPRESS,

EVIDENCIANDO ASÍ SU IMPORTANCIA GLOBAL EN EL MERCADO Y DENTRO DE LAS EMPRESAS.



SEGUNDA NOTA: Por buenas prácticas, cuando un cliente te pida trabajar sobre su página web ya existente,

lo primero a comprobar es si el sitio web está creado con Wordpress o algún otro gestor de contenidos.
